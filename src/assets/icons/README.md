# Add-on Icons

Place your add-on logo/icon files here. Adobe Express requires three sizes:

- `icon-36x36.png` - 36x36 pixels
- `icon-64x64.png` - 64x64 pixels  
- `icon-144x144.png` - 144x144 pixels

All files should be PNG or JPG format.

These icons will be used when you submit your add-on to the Adobe Express Marketplace.
